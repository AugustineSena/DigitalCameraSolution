# DigitalCameraSolution
Using MySQL server example program of campus project using softcomputing algorithm-fuzzy database, fuzzyfy digital camera data. Decision support for Digital Camera through variables :
>1. Price, 
>2. Release date, 
>3. Dimension, 
>4. Weight,
>5. Screen dots,
>6. Max resolution,
>7. Effective pixel,
>8. Photo detectors,
>9. and Sensor size.
